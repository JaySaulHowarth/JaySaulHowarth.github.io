MACHINE LEARNING

This project was completed during my time at QA, the goal was to demonstrate how machine learning can be applied to datasets to exstrapolate a trend and automate processes such as house valuation. While I did have a strong theoretical background in machine learning this was the first time I used tools such as R to apply it to a real business problem. This solution was containerised using docker and hosted on a VM using Google Cloud Platform. Overall I enjoyed the project and it furthered my interest in machine learning.
